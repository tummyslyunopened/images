# images

Shared image assets used across the dotfiles, vendored as a submodule of
[`tummyslyunopened/config`](https://github.com/tummyslyunopened/config).

| File                | Use                                                       |
|---------------------|-----------------------------------------------------------|
| `black.png`         | Solid-black filler / lock-screen background.              |
| `purple-swirl.jpg`  | Generic background image referenced by various configs.   |

For per-host desktop wallpapers see the sibling [`wallpapers`](../wallpapers)
submodule instead.
